## Что мы будем делать в коде

Теперь перейдем к практике. Мы решим **три задачи**, используя Python и GPT. При этом **не напишем ни одной строчки кода вручную** — всё будет генерироваться с помощью GPT.

### Задача 1: Создание Телеграм-бота

- Мы будем использовать библиотеку `aiogram` для создания бота.
- Также будем использовать API OpenAI для генерации ответов.
- Пример кода будет сгенерирован GPT, и мы будем его запускать.
- Если возникнут ошибки — GPT их исправит. Такой подход позволяет быстро идти вперед, не зацикливаясь на деталях.

### Задача 2: Графический интерфейс для нейросети

- Нейросеть будет **определять тональность текста** (позитивная или негативная).
- Мы создадим простой GUI, чтобы удобно взаимодействовать с моделью.
- Всё это делается с помощью GPT, который генерирует код и подсказывает, как его запустить.

### Задача 3: Визуализация датасета

- Мы будем использовать библиотеку `Pandas` и `Matplotlib` для анализа данных.
- GPT поможет сгенерировать код для построения графиков и анализа.
- Это отличный способ понять, как работают библиотеки для анализа данных.
